Search for Cold compound Data for Each cold compound Entity in RDCdb

In the "Cold Compound Search" section, users can search the full-text content of the RDCdb database to identify relevant RDC entries by entering keywords such as RDC drug names, target names, ligand names, linker names, radionuclide names, chelator names, and other related terms.

To perform a search, users simply enter a keyword in the main search box and submit the query. The search results page will display comprehensive information on all RDC drugs directly associated with the search term, including the RDC name, drug development stage, drug type, ligand name, linker name, chelator name, radionuclide name, and detailed links to the corresponding target information.

In order to facilitate a more customized input query, the wild characters of "?" are also supported.

(1) If search "DOTATATE", find a single API entry which is named "DOTATATE";

  1. If search "TATE, finds a series entries with cold compound names.

    For example, if you want to view detailed information about DOTATATE, you can search for "DOTATATE" in the "Cold Compound Search" input box.

Searching for "DOTATATE" returns results that include the "General Cold Compound Information" section. This section displays the General information of the cold compound, including its name, aliases, chemical structure information such as SMILES and formula, physicochemical properties (It should be noted that the physicochemical properties are calculated using the RDKit tool), and external links.

Below the "Full Information of the Activity Data of the RDC(s) Related to This Cold Compound" information list, the radionuclide drug conjugates (RDCs) associated with this cold compound are displayed. The list presents different RDCs along with their development stages.Users can view an overview of activity data at different levels for each RDC, or click the RDC information to quickly navigate to the corresponding RDC activity details page.

The "Related RDCs" section displays the radionuclide-conjugated drugs (RDCs) associated with this cold compound. Clicking each button will redirect users to the corresponding detail page for that radionuclide-conjugated drug.

2.1 By clicking the "RDC Info" button, the detailed information page for particular RDC will be displayed

Using [68Ga]Ga-DOTAGA-TATE as an example, the "General Information of This RDC" section displays the General information of [68Ga]Ga-DOTAGA-TATE, including its ID, drug name, synonyms, drug status, indications, chemical structure, mechanism of action, external links, and other relevant details. In addition, this section also presents General information related to [68Ga]Ga-DOTAGA-TATE, including the cold compound name, ligand name, linker name, chelator name, radionuclide name, and radiochemistry information.

In the "Full List of Activity Data of This RDC" section, three types of activity data are provided: in vitro activity data, in vivo animal data, and human activity data.

Under the "Detailed Activity Data - Human Activity Data" subsection, users can view information on indications, patient characteristics, efficacy data, and related descriptions.

Under the "Animal Activity Data" subsection, imaging or ex vivo biodistribution data are presented, including animal models, cell models, metabolism and excretion information, tumor retention time, therapeutic efficacy, and visualized activity data. For key data, the T/B values are calculated and displayed in tabular format.

The "in vitro data" section presents the lipophilicity coefficient, relevant binding affinity data, and related descriptions.

2.2 By clicking the "Ligand Info" button, the detailed ligand information page for will be displayed.

Using FAP-2286 as an example, the "General Information of this Ligand" section displays the General information of the ligand, including the ligand ID, name, synonyms, external links, chemical structure, molecular formula, SMILES, formula, InChI identifier, InChIKey, IUPAC name, physicochemical properties, and ligand type. It should be noted that the physicochemical properties are calculated using the RDKit tool.

The "Related RDCs" section is about Each Radionuclide Conjugate Related to This ligand,which displays radionuclide drug conjugates (RDCs) with this ligand. Clicking each button will redirect users to the corresponding detail page for that radionuclide drug conjugate (RDC).

The "Complete List of RDCs Associated with This Ligand" section systematically displays the RDCs bound to this ligand. The list presents different types of RDCs along with their structural component information. Users can browse the different RDCs as a whole or click the RDC information to quickly navigate to the corresponding RDC activity page.

2.3 By clicking the "Chelator Info" button, the detailed Chelator information page for will be displayed.

Using DOTA as an example, the "General Information of This Chelator" section presents the General information of the chelator, including its chelator ID, name, synonyms, external links, chemical structure, molecular formula, SMILES, InChI, InChIKey, IUPAC name, physicochemical properties, and ligand type. Please note that the physicochemical properties are calculated using the RDKit tool.

The "related RDCs" section is about Each Radionuclide Conjugate Related to This Chelator, which displays the radionuclide drug conjugates (RDCs) associated with this chelator. Clicking each button will redirect users to the corresponding detail page for that radionuclide drug conjugate (RDC).

2.4 By clicking the "Radionuclide Info" button, the detailed Radionuclide information page for will be displayed.

Using Gallium-68 as an example, the "General Information of This Radionuclide" section presents the General information of the Radionuclide, including its Radionuclide ID, name, synonyms, Radionuclide_half_life, Radionuclide-emission, Radionuclide_energy .

The "Related RDCs" section is about Each Radionuclide Conjugate Related to This Radionuclide, which displays the radionuclide drug conjugates (RDCs) associated with this Radionuclide. Clicking each button will redirect users to the corresponding detail page for that radionuclide drug conjugate (RDC).
